🪶 PANKH

Pankh is a gamified birding companion app built for India’s birdwatching community. It educates about ornithology, preserves local bird lores, connects enthusiasts with real‑time sightings, and makes learning fun through quizzes, challenges, and community engagement.

KEY FEATURES
Gamification
- XP Points & Badges: Earn experience points for birding activities.
- Leaderboards: Compete with friends and the wider community.
- Levels: Progress through birding mastery stages:
- Beginner → Hobbyist → Expert → Purist

Single & Multiplayer Modes: Play solo or challenge friends by joining a Room.

Birding & Community
- Local Bird Lore: Discover cultural stories and traditions tied to Indian birds.
- Recent Sightings: See nearby bird activity sourced from eBird and Cornell Lab datasets.
- Community Hub: Share experiences, sightings, and achievements with fellow birders.

Quizzes & Challenges
- Image Silhouette Quiz: Identify birds by outlines.
- Audio Quiz: Recognize bird calls.
- Birdle: A bird‑themed word puzzle.
- Binocs Spotter: Adjust binoculars to spot hidden birds.

Design System and tokens authored on Figma

Platform and Tools

- Android Studio
- Dart + Flutter Code
- Git and Firebase for cloud
- MongoDB and Hive for local storage
